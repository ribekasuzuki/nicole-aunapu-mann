SOBRE O PROJETO

Este site foi desenvolvido como projeto final da turma 1 do curso "Programação Web para Iniciantes", oferecido pela WOMAKERSCODE em parceria com Potência Tech e Movtech.

Para esta primeira turma, foram selecionadas mais de 600 bolsistas, para garantir que o máximo de mulheres possam ter acesso e ingressar no mundo da tecnologia!

A idea do projeto final é que cada aluna escolhessse uma mulher na área da tecnologia, e para este site a autora buscou por alguém que representasse alguma minoria.

Enquanto buscava inspirações, se deparou com a primeira mulher indígena americana a embarcar em uma missão espacial, e assim a escolhida para o projeto final já estava definida.
